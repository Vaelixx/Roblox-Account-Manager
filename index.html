Features

Everything you need. Nothing you don't.

Built for people who juggle more than one account — without compromising on security or speed.

Encrypted by default

Accounts are stored encrypted on your machine (Windows DPAPI), bound to your Windows profile. Optional master password on top. Cookies never touch disk in plain text — and nothing is ever uploaded.

Multi-instance launching

Run multiple Roblox clients side by side. One-click launch into any game or private server, per account.

Server browser

Browse public servers, filter by region and player count, and join the exact server you want.

Presence tracking

See at a glance which accounts are online, in game, or idle — with grouped views, pins and favorites.

Anti-AFK & watchdog

Keep sessions alive with anti-AFK and let the watchdog restart crashed clients automatically.

Friends & social tools

Friends overview across accounts, presence tracking, and quick actions without switching sessions.

Cookie health checks

Automatic validation keeps sessions fresh and flags dead cookies before they surprise you mid-launch.

Easy import

Import existing accounts in seconds — including migration from other account managers.

Auto-updates

The app keeps itself up to date — with a full changelog before and after every update.

FAQ

Common questions

Is it safe to store my accounts here?
Yes. Cookies are encrypted at rest with Windows DPAPI and bound to your Windows user profile — another user (or another PC) can't decrypt them. An optional master password adds a second layer. Nothing ever leaves your machine.
Does it send my data anywhere?
No. There is no backend and no telemetry. The app only talks to Roblox itself (to launch games and read presence) and to GitHub (to check for updates).
How do multiple instances work?
The app patches the single-instance lock so several Roblox clients can run at once. Each account launches into its own client — join different games or the same server on multiple accounts.
How do updates work?
The app checks GitHub Releases on startup. When a new version is available you'll see a detailed prompt with the changelog; one click downloads and swaps the executable, and after restarting you get a "What's new" summary.
Is it really free?
Yes — free and open source. No paywalls, no premium tier. If you find it useful, a star on GitHub helps.
